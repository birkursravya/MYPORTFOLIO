Projects

Hand Gesture Based Volume Control

This project showcases a real-time gesture recognition system that controls system volume using Python, OpenCV, MediaPipe, and Pycaw. By tracking the distance between fingers, the system allows users to adjust audio levels without physical input devices, demonstrating practical applications of computer vision and AI in human-computer interaction.

ElysiumMind – Emotion Based Media & Self-Love AI

ElysiumMind is an AI-powered mental wellness project designed to enhance emotional awareness and self-care. Using Machine Learning, NLP, and Speech Recognition, it detects a user’s emotions from text and voice inputs and recommends or plays emotion-specific media content. The system supports multilingual emotion recognition and demonstrates the practical applications of AI in mental health, emotion detection, and human-computer interaction.
